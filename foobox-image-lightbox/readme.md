FooBox enables WordPress sites to display images, video, and other HTML content in a responsive, overlay “lightbox” that opens above the page without navigating away. It works out of the box with zero configuration needed, and supports galleries made via WordPress attachments and many popular gallery plugins (e.g. FooGallery, NextGEN, Envira), as well as WooCommerce for product images. The PRO version adds things like social sharing (with deeplinking—so each item can have its own URL), video lightboxes (YouTube, Vimeo, etc.), slideshows including autoplay, multiple themes and color schemes, inline HTML or iframe content, animations, mobile swipe support, Google Analytics integration, and more control over inclusion/exclusion on specific pages. Pricing tiers include monthly, yearly, and lifetime licenses for 1, 5, or 25 sites.
